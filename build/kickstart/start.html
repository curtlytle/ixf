Welcome to IxF KickStart

We are excited that you are going to be using the IxF for your project. A lot of thought and effort have gone in to making this a great solution for you.

To make this easier for you, there are a couple things to make you aware of:

  • Please do not touch anything inside of the ixf folder. It is kept separate from other files so that it can easily be overwritten with newer versions of the Ixf KickStart as they are released.
  • Change the project.original folder name to be just project. (first time only) This is where you will make any edits that are specific to your project. When you get new versions of KickStart the project folder will not be overwritten because the KickStart file contains the project.original named folder. You will be able to safely unzip directly into your project without your project files being lost.
  • Visit http://irinc.github.com/ixf for examples and documentation. This site is where you will always be able to find the latest and greatest examples, documentation, and copies of the IxF.
  • Give us feedback. We want to know how the IxF can help you do your job easier. Please report bugs, enhancement requests, questions, confusions, etc. on github.
