A quick way to visually represent percentages.

This plugin is custom written for the IxF and so this page is the documentation.

For this to work you provide two numbers. Where good starts, and where bad starts. If you said bad=30 and good=70, then 0-30 would be bad, 31-69 would be neutral and 70-100 would be good.

You can reverse it and say good is 30 and bad is 70. Same logic as above, just the good and bad are reversed.

You can set these numbers via global defaults

	ixf.percentages = {
		good: 70,
		bad: 30,
		hideNum:false
	};

or on the percentage element itself
<span class="ixf-percentage" data-percentage-good="30" data-percentage-bad="70">

There is no currently exposed API via the IxF.
