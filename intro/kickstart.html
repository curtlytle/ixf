Start a New Project

Download KickStart 1.1.2

Instructions

  1. Download and put kickstart folder where you want it.
  2. In the "project/includes" directory, modify "initMenu.php" with your desired menu structure.
  3. Use the "/project" directory for all your project specific JavaScript and CSS.
  4. The "/ixf" directory should never be touched. This allows you to easily update it with the latest version of the IxF very easily.
  5. Take a half hour break because the IxF code just saved you at least that and because there are some really funny baby videos on YouTube.

Launch the "Using KickStart" screencast
