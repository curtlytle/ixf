KickStart

Start a New Project

Download KickStart 1.1.2
(full IxF demo site kickstart download)

Instructions

  1. Download and put kickstart folder where you want it.
  2. In the "project/includes" directory, modify "initMenu.php" with your desired menu structure.
  3. Use the "/project" directory for all your project specific JavaScript and CSS.
  4. The "/ixf" directory should never be touched. This allows you to easily update it with the latest version of the IxF very easily.
  5. Take a half hour break because the IxF code just saved you at least that and because there are some really funny baby videos on YouTube.

Launch the "Using KickStart" screencast

Production Code

Use the following code:

<!-- BEGIN IXF ASSETS -->
<link rel="stylesheet" type="text/css" media="all" href="ixf/styles/screen.css" />
<!--[if lte IE 7]>
<link rel="stylesheet" type="text/css" media="all" href="ixf/styles/screen-ie.css"/>
<![endif]-->
<script type="text/javascript" src="ixf/scripts/jquery.min.js"></script>
<script type="text/javascript" src="ixf/scripts/jquery-ui.min.js"></script>
<script type="text/javascript" src="ixf/scripts/ixf-plugins.min.js"></script>
<script type="text/javascript" src="ixf/scripts/ixf-utilities.min.js"></script>
<!-- END IXF ASSETS -->
