Themeroller

Pick a color, any color. (please don't actually pick any of those colors)

Not all of them will look great. But hey, at least it's at least a place to start.

The IxF uses lots of CSS gradients to reduce image requirements. Because of that it is easy to update the theme of the site via CSS as well. This page allows you to pick a base color and then we create the lighter and darker colors for you. Yes, we just like you that much.

After finding your happy color, click the button below to get the CSS that you should put in your project.css. The URL for this page is updated to reflect whatever color you have chosen. So you can bookmark or send it to that special someone.

Get your custom CSS

Things to know

  • All this does it update the outer chrome colors and the popups. The colors for components like forms, buttons, tabs, etc are fairly neutral and will work with many colors. If you need those to change, you are on your own.
  • The pointer part of the popups will not update colors on the fly like everything else. They are built using canvas and is based off the color of popup border when the popup first opens. To see them update as well you will need to refresh.
  • You will need to change the color of the little circle in the top right of the nav yourself. We wanted to keep this simple and have you only pick one color.
